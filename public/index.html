The Modular Ecosystem

Build your own nodes.

Jenn is built on a plug-and-play architecture. Anyone can build their own input or output nodes using the Jenn API. Our developer-first approach ensures that your workflow is never limited by our integrations.

node_definition.ts
export const myCustomNode = describe({
  id: "custom-input",
  type: "SOURCE",
  schema: { ... },
  handler: async (data) => {
    // Your logic here
  }
});
