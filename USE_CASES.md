# Jenn — сценарии использования

КРУТО: Расширение для браузера - аналог закладок многие люди просто сохраняют сайт и никогда  его не открывают а теперь он будет отправлять в какую-либо страницу Notion
Удобно вести таблицу доход расход Jenn четко все запищет


## 🧠 Для кого это вообще

Jenn решает проблему «информационного шума» и «выбора куда сохранить».  
Обычный человек за день видит десятки ссылок, мыслей, задач — и большинство теряется.  
Jenn — это **единое окно ввода**: сказал → Jenn сам разобрался куда.

---

## 1. Личная продуктивность

### 1.1. Захват идей (откуда угодно)

Гуляешь, пришла мысль → открыл Telegram → написал боту:  
> «идея: сделать приложение для поиска попутчиков»

**Jenn:**  
- распознал `create_note`
- определил категорию `Идеи`
- сохранил в Notion → база «Идеи для стартапов»

> *Раньше:* открыть Notion → найти базу → создать страницу → придумать название  
> *С Jenn:* одно сообщение в Telegram

### 1.2. Сбор ссылок/контента

Переслал пост из канала про нейросети:
> https://t.me/ai_news/1234

**Jenn:**  
- распознал `save_entry`  
- AI прочитал заголовок → категория `Нейросети`  
- сохранил ссылку в Notion → база «Статьи по AI»  
- и отдельно добавил в Google Sheets → таблица «Контент-план»

> *Раньше:* скопировать ссылку → открыть закладки → найти папку → вставить  
> *С Jenn:* переслал в бота → готово

### 1.3. Напоминания и задачи

> «напомни купить подарок маме до пятницы»

**Jenn:**  
- распознал `create_reminder`  
- дата: пятница  
- отправил в Telegram → придёт уведомление

> *Раньше:* открыть напоминалку → создать → заполнить дату, время, текст  
> *С Jenn:* сказал и забыл

### 1.4. Ежедневный дамп (dumping)

В конце дня открыл desktop input (Alt+Space) и записал:
> «встреча с клиентом прошла хорошо, но нужно доработать презентацию, ещё позвонил сантехник сказал приедет в среду»

**Jenn:**  
- «встреча с клиентом» → `create_note` → Notion → категория «Работа»
- «позвонил сантехник» → `create_reminder` → Telegram → среда
- «доработать презентацию» → `create_task` → Todoist

> *Раньше:* 3 разных действия в 3 разных сервисах  
> *С Jenn:* один поток сознания → распределился сам

## 2. Работа с командой (если добавить workspace)

### 2.1. Сквозные заметки

Написал в Jenn:
> «результаты спринта за эту неделю — выкатили авторизацию, починили баг с датами, в пятницу ретро»

**Jenn:**  
- `create_note` → Notion → база «Командные заметки»  
- доступно всей команде

### 2.2. Баг-трекер через Telegram

QA-инженер переслал скриншот с ошибкой:
> «баг: при нажатии на кнопку сохранить падает с 500»

**Jenn:**  
- `create_task` → GitHub Issues / Linear  
- приоритет: high  
- назначить на тимлида

---

## 3. Контент-мейкер / маркетолог

### 3.1. Сбор материалов для постов

В течение дня попадаются: статья, мем, цитата, новость.  
Всё кидается в один Telegram-чат с ботом.

Вечером открываешь — а там уже разложено:
- **Статьи** → Notion → «Для блога»
- **Мемы** → Google Drive → «Визуал»
- **Цитаты** → Notion → «Идеи для постов»
- **Новости** → Notion → «Контент-план»

### 3.2. Ресерв тем для видео

> «сделать видео про нейросети для дизайнеров — показать Midjourney, Stable Diffusion, подводные камни копирайта»

**Jenn:**  
- `save_entry` → Notion → база «Идеи для видео»  
- категория: AI  
- AI сам добавил тэги: {нейросети, дизайн, контент}

## 4. Студент / исследователь

Читаешь статью → выделил ключевой абзац → отправил в Jenn:

> «Важно: исследование показало что модели на 30% эффективнее при fine-tuning на специфических данных»

**Jenn:**  
- `save_entry` → Notion → база «Диплом / Исследование»  
- AI автоматически поставил категорию `Машинное обучение`  
- добавил в список источников

> *Раньше:* копировать → открыть Notion → найти где-то вкладку → вставить — и это на каждую цитату  
> *С Jenn:* выделил → отправил → готово

---

## 5. Предприниматель / фрилансер

### 5.1. Управление заказами

> «клиент попросил сделать лендинг за 2 недели, бюджет 50к»

**Jenn:**  
- `create_task` → Notion → база «Заказы»  
- `create_event` → Google Calendar → дедлайн через 2 недели  
- `create_note` → Notion → «Переговоры» → запись о клиенте

### 5.2. Учет расходов

> «купил хостинг 1500р, подписка на AI 20$»

**Jenn:**  
- `save_entry` → Google Sheets → таблица «Расходы»  
- категория: AI → строка добавилась автоматически  
- в конце месяца не надо ничего вспоминать

## 6. Семья / быт

> «закончился стиральный порошок, вечером забрать посылку с почты, в субботу ужин у родителей в 18:00»

**Jenn:**  
- «порошок» → список покупок (Google Keep)  
- «посылка» → напоминание (Telegram)  
- «ужин» → календарь (Google Calendar)

Всё с одного сообщения в семейный чат Telegram.

---

## 7. Сценарий "Буфер" (всё что не распознано)

Если Jenn не смог определить категорию → всё падает в **буфер**.

Утром пользователь открывает буфер:
> «тут 5 сообщений. Перенести "ссылку на курс" → Обучение, "мем с котом" → Развлечения, "идея приложения" → Идеи»

Научился — в следующий раз AI сам определит эти категории.

---

## Главное

**Jenn — это не замена Notion, Google Sheets или Telegram.**

Это **исчезающий слой** между человеком и этими сервисами.  
Человек не думает о том, в какой tool что класть. Он просто **говорит** — и оно оказывается там, где надо. Автоматически.

Именно это отличает Jenn от любого существующего решения:  
**AI определяет намерение → классифицирует → раскладывает по нужным сервисам.**  
Ни один инструмент на рынке не делает это в self-hosted формате с привязкой к Notion, Sheets, Obsidian и т.д.
